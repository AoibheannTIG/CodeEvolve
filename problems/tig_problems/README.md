# TIG Problems

TIG (The Innovation Game) problems are run from the **tig-evolve** repository, not from this directory.

## Getting Started

### Clone the tig-evolve Repository

```bash
git clone https://github.com/tig-foundation/tig-evolve.git
cd tig-evolve
```

### Available Examples

The tig-evolve repo contains examples for running TIG challenges with different evolution frameworks:

- **OpenEvolve**: `examples/openevolve/`
- **CodeEvolve**: `examples/science-codeevolve/`


### Running with CodeEvolve

```bash
cd tig-evolve
export API_KEY="your-api-key"
bash examples/science-codeevolve/run.sh
```
